# MFE - ESLint Shareable Config

## Install

```bash
yarn add eslint-config-hfe --dev
```

## Usage

add this to your .eslintrc file:

```
{
  "extends": "eslint-config-mfe"
}
```

You can override settings from the shareable config by adding them directly into your
`.eslintrc` file.
